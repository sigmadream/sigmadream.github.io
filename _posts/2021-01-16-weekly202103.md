## Python

- [A Visual Intro to NumPy and Data Representation](http://jalammar.github.io/visual-numpy/)

  - 데이터분석, ML 그리고 딥러닝을 아우르는 핵심 라이브러리인 NumpPy에 대한 가벼운 사용벙법과 데이터 유형을 소개하는 기사입니다.
  - 자주 접하는 기사라 할 수 있겠으나, 거의 모든 과정을 시각적으로 알려주고 있기 때문에 구체적인 예제가 목말랐던 분들에게 좋은 기사입니다.

- [Hacking QR code design](https://marienraat.nl/hacking-qr-codes.html)
  - QR코드를 원하는 디자인에 맞춰서 작성하는 방법을 소개하는 기사입니다.
  - 해당 기사는 QR 코드의 데이터를 기반으로 디자인하지 않고, 디자인에 맞춰서 QR 코드를 생성하는 방법을 소개합니다.
  - 관점의 차이를 기반으로 한 멋진 아이디어를 실현하는 과정을 소개하는 기사입니다. QR 코드가 아니더라도, 데이터 시각화에 관심이 있으시면 읽어두시면 좋을 기사입니다.

## JavaScript

- [Video Chatting and Screen Sharing with React, Node, WebRTC(peerjs)](https://dev.to/arjhun777/video-chatting-and-screen-sharing-with-react-node-webrtc-peerjs-18fg)

  - 아주 간단한 화상회의 기능을 만들어 볼 수 있는 튜토리얼 기반의 기사 입니다.
  - 대부분의 코드를 Github으로 제공하고 있기 때문에, 가볍게 도전해보시기 좋은 튜토리얼입니다.

- [Use Node.js with Docker and Docker Compose to improve DX](https://blog.logrocket.com/node-js-docker-improve-dx/)
  - Docker와 Node.js를 사용하여 개발자 환경을 개선하는 방법을 소개하는 기사 입니다.
  - Docker 빌드를 효율적으로 사용하고 Docker Compose를 활용하여 개발 환경을 구성하는 방법을 튜토리얼 형식으로 자세히 소개하고 있습니다.

## Mobile

- [Using and testing Room Kotlin APIs](https://medium.com/androiddevelopers/using-and-testing-room-kotlin-apis-4d69438f9334)

  - Room Kotlin API를 사용하고 테스트하는 방법을 소개하는 기사입니다.

- [Understanding of Property Wrappers in SwiftUI](https://medium.com/mindful-engineering/understanding-of-property-wrappers-in-swiftui-3789a72515c0)
  - SwiftUI가 효율성을 위해서 많이 활용하는 속성 래퍼(property wrappers)에 대해서 소개하는 기사입니다.
  - SwiftUI를 시작하신다면 읽어두시면 좋을 기사입니다.

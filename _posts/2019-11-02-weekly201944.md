## Python

- [Meta-Programming in Python](https://medium.com/better-programming/meta-programming-in-python-7fb94c8c7152)

  - Pyhon의 선뜻 손이 가지 않는 Decorators와 Meta-classes를 설명하고 있으며, 한 걸음 더 나아가 Meta-Programming에 대해서 간략하게 소개하는 기사 입니다.
  - 평소에 `@something`이 궁금하셨다면 이 기사를 참고해 보세요!

- [Binning Data with Pandas qcut and cut](https://pbpython.com/pandas-qcut-cut.html)

  - Pandas에서 변수 구간화가 궁금하셨다면 이 기사를 읽고 연습해보세요!
  - 이산형 변수를 범주형 변수로 변환하는 것을 일반적으로 변수 구간화(Binning)라 하는데, 데이터를 정리하다 보면 생각보다 많은 곳에서 사용하니 연습해 두면 데이터 정리할 때 많은 도움이 됩니다.

- [All You Need to Know About Big O Notation [Python Examples]](https://skerritt.blog/big-o/)

  - Python 코드로 시간 복잡도를 아름답고 상세하게 설명하고 있는 이번주 강추 기사 입니다.
  - 함께 읽으면 좋은 유사한 기사로 [Divide and Conquer Algorithms with Python Examples](https://skerritt.blog/divide-and-conquer-algorithms/)도 함께 읽어보세요!

- [Complete Guide to Imports in Python: Absolute, Relative, and More](https://www.pythonforthelab.com/blog/complete-guide-to-imports-in-python-absolute-relative-and-more/)
  - 별것 아닌 것 같지만 알아두면 많은 도움이 되는 `Import`에 관한 상세한 튜토리얼 기사 입니다.

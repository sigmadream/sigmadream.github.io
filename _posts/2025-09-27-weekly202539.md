## C\# and JVM

- [eShop Reference Application - "AdventureWorks"](https://github.com/dotnet/eShop)
    - .NET Aspire 기반 서비스 아키텍처로 설계된 모범적인 e-commerce 참조 구현 애플리케이션 입니다. .NET 9 기반이며, 프런트/백엔드 분리, Docker 및 Azure OpenAI 연동 등 최신 .NET API 및 Azure 통합 사례 제공.
    Windows(Visual Studio)와 Mac/Linux 모두 지원하며, PowerShell 또는 Azure Dev CLI 등 다양한 방식의 환경 자동 설정을 안내합니다. 참고로 샘플 데이터는 GPT-3.5-Turbo와 DALL·E 3로 생성되었습니다. 무엇보다 실제 배포형 클라우드/온프렘 환경을 염두에 둔 구성 및 문서화가 우수함.

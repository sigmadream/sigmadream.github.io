## Python

- [Python Datetime Tutorial: Manipulate Times, Dates, and Time Spans](https://www.dataquest.io/blog/python-datetime-tutorial/)

  - Python에서 `datetime` 및 `calender` 모듈을 사용하는 방법을 자세히 소개하는 기사 입니다.
  - 이와 관련 기사로 [Stop using utcnow and utcfromtimestamp](https://blog.ganssle.io/articles/2019/11/utcnow.html)도 함께 참고하세요. `utcnow`의 위험성을 설명하고 대체할 수 있는 방법 중 하나인 `fromtimestamp`를 사용해야 하는 이유를 설명하는 기사입니다.

- [JWT Auth with DjangoREST API](https://medium.com/swlh/jwt-auth-with-djangorest-api-9fb32b99b33c)

  - 토큰기반 인증에 널리 사용되는 JWT(JSON Web Tokens)를 Django REST Framework에서 사용하는 방법은 간단한 튜토리얼을 통해서 설명하는 기사입니다.

- [Less known bits of the Python Standard Library](https://medium.com/@vmsp/less-known-bits-of-the-python-standard-library-46dc88490115)
  - Python의 유용하지만 덜 알려진 표준 라이브러리에 대한 소개입니다(모르고 지나쳤던 좋은 라이브러리가 많으니 시간되시면 참고하세요!)

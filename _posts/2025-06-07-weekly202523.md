## Python

- [Traffic meter per ASN without logs](https://anarc.at/blog/2025-05-30-asncounter/)
  - 이 기사는 네트워크 트래픽을 ASN(Autonomous System Number) 단위로 측정하는 방법을 소개합니다. 로그 파일을 저장하지 않고도 실시간으로 트래픽을 집계하는 방법, 필요한 도구와 설정 과정, 그리고 실제 운영 환경에서의 활용 예시를 다룹니다. 또한, 프라이버시 보호와 효율적인 네트워크 관리라는 두 가지 측면에서 이 방법의 장점을 설명합니다.
  - 로그를 남기지 않기 때문에 프라이버시 측면에서 눈여겨 볼 장점을 가진 기사입니다.


## JavaScript

- [Why React error boundaries aren't just try/catch for components](https://www.epicreact.dev/why-react-error-boundaries-arent-just-try-catch-for-components-i6e2l)
  - 이 기사는 React의 에러 바운더리(Error Boundary)가 단순히 컴포넌트 내부에서 발생한 예외를 try/catch로 처리하는 것과 어떻게 다른지 설명합니다. 에러 바운더리는 렌더링, 라이프사이클 메서드, 그리고 자식 컴포넌트 트리에서 발생하는 예외를 포착해 UI 전체가 무너지는 것을 방지하며, 일반적인 try/catch로는 잡을 수 없는 React 렌더링 과정의 오류를 효과적으로 처리할 수 있습니다. 또한, 에러 바운더리의 동작 원리와 실제 사용 예시, 그리고 올바른 에러 핸들링 전략에 대해 다룹니다.

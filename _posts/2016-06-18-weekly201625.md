## Python with PY Family

- [Web Service Efficiency at Instagram with Python](https://engineering.instagram.com/web-service-efficiency-at-instagram-with-python-4976d078e366)

  - 파이썬 장고(Django) 프레임워크를 사용하고 있는 인스타그램에서 웹 서비스를 효율적으로 운영하는 하는 방법을 소개하고 있음
  - 다른 어떤 챕터보다 모니터링/알림에 관련된 매커니즘 부분은 팀원들과 같이 도란 도란 이야기해 볼 필요가 있음

- [How to build and deploy a Facebook Messenger bot with Python and Flask, a tutorial](http://tsaprailis.com/2016/06/02/How-to-build-and-deploy-a-Facebook-Messenger-bot-with-Python-and-Flask-a-tutorial/)

  - 파이썬 Flask를 사용해서 페이스북 메시젼 봇을 만드는 방법을 소개하고 있음
  - Flask를 사용하고 있는 좋은 예제라 생각해서 진행해 보았고, 실행도 잘되므로 Flask 기반의 좋은 예제 중 하나라 할 수 있음

- [Automation for the People](https://gist.github.com/classam/9e07a36aa63624ca2dda75a1367a53c6)
  - 자동화와 관련된 몇가지 기술에 대해서 소개하고 있음
  - 자동화에 관련된 실수를 이야기하고 있기 때문에 꼭 읽어볼 필요가 있음

## Android with Google Family

- [Use Android Studio Like a Pro!](https://stanfy.com/blog/use-android-studio-like-a-pro)

  - Android Studio를 잘 사용하기 위한 몇가지 팁(Tip)을 소개하고 있음
  - '검색 도구(Action search)', '멀티커서(Multicursor)', 'Groovy Console', 'Logcat' 부분은 잘 읽어보면 충분히 매력적임

- [Clean Java immutability](http://blog.alexsimo.com/clean-java-immutability)

  - 불변 클래스(immutability class)를 소개하고, 장점을 일반 자바 코드 및 라이브러리 활용등을 통해서 선보이고 있음
  - 문자(String)의 '불변'에 대한 약간의 선입견이 있었는데 이 글 덕분에 새로운 견해를 가질 수 있었음

- [No, You Can Not Override the Home Button... But You Don't Have To!](http://www.developerphil.com/no-you-can-not-override-the-home-button-but-you-dont-have-to/)
  - 안드로이드 홈(home) 버튼을 재정의하고 싶어하는 개발자의 욕망인 '앱 사용 중지'시 개발자에기 필요한 몇가지(백그라운드 진입시 알림 등...)에 대해서 소개하고 있음
  - 현재 만들고 있는 앱의 설계와 사용자 UX에 관련된 프로세스를 좀 더 면밀히 살펴볼 필요가 있을 것 같아서 공유해서 읽었음

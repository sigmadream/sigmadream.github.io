## Python

- [Machine Learning Prototyping with DuckDB and scikit-learn](https://duckdb.org/2025/05/16/scikit-learn-duckdb.html)
  - 이 기사는 DuckDB와 scikit-learn을 활용해 머신러닝 프로토타입을 빠르게 개발하는 방법을 소개하고 있습니다. - 이 기사는 DuckDB에 입문하시기 좋은 글입니다.

- [Understanding Random Forest using Python (scikit-learn)](https://towardsdatascience.com/understanding-random-forest-using-python-scikit-learn/)
  - 이 기사은 scikit-learn을 활용해 파이썬에서 랜덤 포레스트(Random Forest) 알고리즘의 원리와 동작 방식을 쉽게 이해할 수 있도록 설명하며, 랜덤 포레스트의 기본 개념, 장점, 실제 코드 예제와 함께 각 단계별로 어떻게 모델이 학습되고 예측하는지 자세히 다룹니다.

## JavaScript

- [Concurrent Rendering](https://ui.dev/c/react/concurrent-rendering)
  - 이 기사는 React의 Concurrent Rendering(동시 렌더링) 개념을 소개하며, 기존 렌더링 방식과의 차이점, 동시성의 필요성, 그리고 React가 어떻게 더 부드럽고 응답성 높은 UI를 제공하는지 설명합니다. 또한, 동시 렌더링이 실제로 어떻게 동작하는지 예시와 함께 직관적으로 이해할 수 있도록 도와줍니다.
  - Concurrent Rendering은 React가 여러 작업을 동시에 처리해 UI의 응답성을 높이는 기술입니다. 이를 통해 사용자는 더 빠르고 부드러운 인터페이스를 경험할 수 있을 것으로 예상됩니다. 관련해서 읽어보시면 참고가 될 기사입니다.

- [Build Your Own TanStack Query](https://mugglim.github.io/build-your-own-tanstack-query/)
  - 이 글은 TanStack Query(구 React Query)와 같은 데이터 패칭 라이브러리의 핵심 원리를 직접 구현해보는 과정을 단계별로 설명합니다. 저자는 캐싱, 쿼리 무효화, 자동 리패칭 등 TanStack Query의 주요 기능을 직접 만들어보며, 각 기능이 어떻게 동작하는지 코드와 함께 상세히 다룹니다. 이를 통해 TanStack Query의 내부 구조와 동작 원리를 깊이 있게 이해할 수 있습니다.
  - TanStack Query의 핵심 기능(캐싱, 쿼리 무효화 등)을 직접 구현하며 동작 원리를 배울 수 있습니다. 실제 라이브러리의 내부 구조를 코드로 따라가며 이해하고 싶은 분께 추천합니다.
